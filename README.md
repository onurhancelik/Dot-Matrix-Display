# Dot-Matrix-Display
Embedded and Real Time Systems - Term Project - Embedded C

The aim of this project is displaying characters that are received by keyboard from user on the LED Display Driver. 
Characters can be any key on the keyboard including Turkish characters.

In this project, we used Stellaris LaunchPad LM4F120 of Texas Instruments that is include Cortex M4 processor and MAX7219 LED Display Driver.
We wrote embedded C language on Energia platform and also we made user interface to get keyboard inputs using MS Visual Studio 2015 - C# language.
We used Serial Port COM4 to communicate LaunchPad with the keyboard.
